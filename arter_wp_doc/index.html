Arter - Theme Documentation

Thank you for purchasing our theme!
This guide provides instructions for installation. It is recommended you read it thoroughly to fully leverage the "Arter" theme's capabilities. If you need any help or support please contact us through the ThemeForest Messaging system or Support Email: beshleyua@gmail.com

Installing WordPress

Before beginning the installation and configuring of your new theme, you must first have WordPress already installed on a server. Here is a very detailed guide at Wordpress Codex that will walk you through every step.

Installing Theme

Uploading Through WordPress

To upload through your Dashboard, select “Appearance” and then “Themes”. Next, select the “Add New” button at the top of the page and then select the “Upload Theme” button. Choose Arter theme "arter.zip" file and select “Install Now”. Once the theme is fully uploaded and installed, click “Activate” to activate the WordPress theme.

When uploading your theme with the installer, please ensure you are uploading the theme .zip file, not the entire package you downloaded. In this case, you will be uploading "arter.zip".

FTP Upload

To manually upload your new WordPress theme, login with your credentials to your website and locate the wp-content folder in your WordPress install files. Upload the un-zipped Arter WordPress Theme folder into the: wp-content/themes folder.
Once uploaded, activate the theme by heading to the “Themes” menu in the WordPress Dashboard.

Installing Plugins Required

Once you have installed and activated our theme you will be prompted to install required and recommended plugins for all theme features. Just click the link to Begin installing plugins and activate all plugins.

Elementor Builder

- To enable on your page, please do the following:
1. Go to Dashboard > Pages > Add New (or Edit).
3. In the tab "Page Attributes" > "Template", select "Layout builder (Elementor)" and Save Changes.
4. Click to "Edit with Elementor" button.
5. In Elementor builder panel drag and drop "Arter Theme" elements & blocks and fill the fields.

Theme Options

All global settings can be found in ​Dashboard > Theme Options​. In addition, you can customize:

Styling

In "Styling" Tab you can edit: Page Transition (Enable Smooth page transitions), Theme Color, Heading Color, Text Color, Menu Items Color, Heading Size, Paragraph Size, Menu Items Size, Heading Font, Menu Items Font, Paragraphs Font.

Header & vCard

In "vCard" Tab you can edit: Header Background, Profile Image, Profile Name, Profile Short Description, Profile Information, Profile Skills Blocks, Profile Buttons.

Footer

In "Footer" Tab you can edit: Footer Text, Footer Text (Right), Socials buttons.

Blog

In "Blog" Tab you can edit: Blog page, enable/disable Categories, disable/enabled Post Excerpt, disable/enabled Featured Image, Social Share.

Portfolio

In "Portfolio" Tab you can edit: Portfolio Page, Disable Lightbox (on portfolio).

404

In "404" Tab you can edit: Title 404, Content 404.

Preloader

In "Preloader" Tab you can edit: Preloader Text.

2. Enter your Page Title "Home".
3. In the tab "Page Attributes" > "Template", select "Layout builder (Elementor)". In the tab "Publish" click to "Publish" button.
4. Click to "Edit with Elementor" button.
5. In Elementor builder panel drag and drop "Arter Theme" elements: "Hero Banner", "Numbers", "Portfolio", "Pricing", "Project Banner", "Project Description", "Project Image Carousel", "Blog (Carousel)", "Blog (Grid)", "Clients", "Contact Form", "Contact Information", "Project Image", "Project Image Gallery", "Resume", "Services", "Testimonials" and more.

Contact Form

- To create Contacts Form, please do the following:
1. Go to Dashboard > Contact > Contact Forms > Add New.
2. Enter form name "Contact Form".
3. In the tab "Form" you can use default form layout or enter this code lines:

<!-- contact form -->
<div class="art-contact-form">
	<!-- form field -->
	<div class="art-form-field">
		<!-- name input -->
		[text* your-name class:art-input placeholder "Name"]
		<!-- label -->
		<label><i class="fas fa-user"></i></label>
	</div>
	<!-- form field end -->
	
	<!-- form field -->
	<div class="art-form-field">
		<!-- email input -->
		[email* your-email class:art-input placeholder "Email"]
		<!-- label -->
		<label><i class="fas fa-at"></i></label>
	</div>
	<!-- form field end -->
	
	<!-- form field -->
	<div class="art-form-field">
		<!-- message textarea -->
		[textarea* your-message class:art-input placeholder "Message"]
		<!-- label -->
		<label><i class="far fa-envelope"></i></label>
	</div>
	<!-- form field end -->
	
	<!-- button -->
	<div class="art-submit-frame">
		<button class="art-btn art-btn-md art-submit" type="submit"><span>Send message</span></button>
	</div>
</div>
<!-- contact form end -->

4. In the tab "Mail", enter this your email address.
5. Save Changes.

Create Menu

- To create Create/Edit Menu, please do the following:
1. Go to Dashboard > Appearance > Menus.
2. Select create a new menu at the top of the page.
3. Enter a name for your new menu in the Menu Name box.
4. Click the Create Menu button.
5. Select the Pages that you want to add by clicking the checkbox next to each Page's title. Click the Add to Menu button.

Update Our Themes

Update via Envato Market Plugin

To easy and fast update theme and plugins you can use Envato Market Plugin (I added this plugin in recommended list in the theme):
1) Install Envato Market Plugin: https://envato.com/market-plugin/ and activate.
2) On admin panel go to Envato Market -> Settings -> Generate an Envato API Personal Token -> Click on link -> Login to your Envato Account -> Checked “Permissions ( https://gyazo.com/8a94c801e8197207d3d37467aa8587fb )” on list -> Create Token.
3) Use plugin “Envato Market -> Themes” to install all future updates. (in a few clicks).
Thanks!

Manually Update (Appearance -> Themes)

You can update our theme via WordPress Panel, Go to Dashboard > Appearance > Themes. Click "Add New" -> "Upload Theme". Choose Arter theme "arter.zip" file and select “Install Now”.

Manually Update (FTP or Cpanel)

You can update our theme using via FTP or Cpanel.
Download the latest version of the theme on ThemeForest download page and you rename the folder contain the old theme and create a new folder and insert the latest theme in here.
